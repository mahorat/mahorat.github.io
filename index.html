Create Post

The posts API allows accounts to store posts to Babolo Jobs website. Creating posts possible only after the user authentification.

The API supports a variety of post types: vacancy, article, company and resume. Specific post data examples and required parameters for each post type are given below.

Validation

Maximum total JSON size of the request is 30kb.

Resource URL

General create post parameters

The following parameters are available for all post types:

Name Description Validation
title The post title to display required, string, min:5, max:255
excerpt The post excerpt to display optional, string, min:10, max:500
content The post content to display optional, string, min:50, max:5000
image The post image to display optional. Image size should be no more than 4 MB. Recommended 720x720
type Post type required. Available message types: company, vacancy, resume and article

Getting posts

The posts API allows to get posts from Babolo Jobs website. Getting posts possible for each users with or without authentification.

The API supports a variety of post types: vacancy, article, company and resume. Specific get data examples and required parameters for each post type are given below.

Resource URL

GET parameters

This API method uses the same parameters as the request GET method with a few variations described below.

Name Description Validation
type Post type required. Types: company, vacancy, resume, article
category Posts with category optional, array
tags The posts of tagged in optional, array
